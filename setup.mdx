## Prerequisites

Before you begin, ensure your environment meets the following requirements:

  - **Python 3.8 or newer**: PureRAG is compatible with the latest versions of Python.
  - **pip**: Ensure pip is installed and updated to the latest version.
  - **Virtual Environment (optional)**: Using a virtual environment such as `venv` or `conda` is recommended for dependency management.

To update pip, run:
```bash
pip install --upgrade pip
```

## Installation
Installing PureRAG is as simple as running a single pip command. Open your terminal and execute:

```bash
pip install purerag
```

### Verify Installation
After installation, you can verify that PureRAG is installed by running the following in your terminal:

```bash
python -m pip show purerag
```
If the package details are displayed, the installation was successful.
